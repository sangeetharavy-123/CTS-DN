# Singleton Pattern in Java

## Overview
This project demonstrates the Singleton Design Pattern using a Logger class. It ensures that only one instance of the class is created and shared throughout the application.

## Components
- `Logger.java` – Implements the Singleton pattern.
- `Main.java` – Tests and verifies that only one object is created.

## Implementation
- Private static instance variable
- Private constructor
- Public `getInstance()` method

## Key Concept
The Singleton pattern ensures that a class has only one object and provides a global access point to it.
