# Exercise 7: Financial Forecasting

## Objective
Predict future financial values using a recursive algorithm.

## Concept
Recursion is a technique where a method calls itself until a base condition is reached.

## Formula
Future Value = Present Value × (1 + Growth Rate)^Years

## Input
- Present Value
- Growth Rate
- Number of Years

## Output
- Future Value after the specified number of years

## Time Complexity
O(n)

## Space Complexity
O(n)

## Optimization
The recursive solution can be optimized using:
- Iterative approach (for loop)
- Dynamic Programming
- Memoization

## Conclusion
Recursion helps calculate future values by repeatedly applying the growth rate for each year. It is easy to understand but uses additional stack memory compared to an iterative solution.
